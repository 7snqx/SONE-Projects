# Symulator Bramek Logicznych

Interaktywna aplikacja webowa do nauki i symulacji bramek logicznych. Pozwala użytkownikowi przeciągać bramki na kanwę, łączyć je ze sobą i testować przepływ sygnałów logicznych w czasie rzeczywistym.

## Funkcjonalności

- **11 Typów Bramek**:
  - Źródła: INPUT, OUTPUT
  - Podstawowe: AND, OR, NOT
  - Rozszerzone: NAND, NOR, XOR, XNOR
  - Sumatory: HALF ADDER, FULL ADDER
- **Drag & Drop**: Intuicyjny interfejs przeciągnij i upuść.
- **Symulacja w czasie rzeczywistym**: Natychmiastowa wizualizacja przepływu sygnału.
- **Wizualizacja**: Aktywne połączenia i porty podświetlają się na zielono.

## Uruchomienie

Wystarczy otworzyć plik `index.html` w dowolnej nowoczesnej przeglądarce internetowej. Aplikacja nie wymaga procesu budowania (korzysta z Tailwind CSS przez CDN).

## Technologie

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6)
